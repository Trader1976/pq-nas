Uploads & Tiering Help
Upload limits


Upload tiering
New uploads land in a fast internal pool first and later migrate to capacity storage.

USB / removable pools are automatically blocked for safety.

Worker interval
How often the migration worker wakes up and scans landing storage.

Minimum landing age
A file must remain in landing at least this long before migration is allowed.

Max files per pass
Maximum number of eligible files migrated during one worker scan.

Example:
interval = 60
min age = 60
max/pass = 8

Worker wakes every 60 seconds, migrates files older than 60 seconds, and processes up to 8 files each cycle.
