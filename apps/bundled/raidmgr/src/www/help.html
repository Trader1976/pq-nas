Storage Manager Help

Manage storage pools, slots, drives, and RAID mode in PQ-NAS.

What this page is for

Storage Manager is used to create and maintain storage pools in PQ-NAS. It shows all pools, their slots, current member drives, usage summary, and management actions.

Main actions

  • Create new pool creates a new editable PQ-NAS pool.
  • Edit slots changes the saved slot layout and pool mode metadata.
  • Add slot adds one more empty visible slot.
  • Remove empty slot removes one unused empty slot.
  • Apply layout applies one pending membership change from the saved slot layout to the real Btrfs pool.
  • Remove drive removes a current member drive from the pool.
  • Rename changes the display name of the pool.
  • Convert RAID converts between single and raid1.
  • Destroy removes the pool from PQ-NAS and can optionally wipe member drives.

Slots vs member drives

Slots are the saved PQ-NAS layout. Member drives are the drives currently attached to the live Btrfs pool.

These can temporarily differ. When they differ, Pool Manager shows a pending layout state. Use Apply layout to make the real pool membership match the saved layout.

Single and RAID1

Single Capacity-focused mode with no redundancy.

RAID1 Mirror-focused mode with redundancy across at least two devices.

Converting RAID mode may run a Btrfs balance and can take a long time depending on pool size and device speed.

Important warnings

Long-running operations Add, remove, and convert operations may take a long time.

Preview first When an action uses Preview and Apply, always preview before applying.

Destructive wipe Force wipe or destroy with wipe enabled can permanently erase data.

Be careful with old Btrfs signatures A disk that still contains old Btrfs metadata may look like a pool member even when it is no longer active.

Recommended workflow

  1. Create a pool with the desired initial mode.
  2. Adjust slot count if needed.
  3. Assign drives or add/remove them through the pool actions.
  4. Use Apply layout when saved slots and live membership differ.
  5. Use Convert RAID only when the current device count supports the target mode.
